

Once inside the main folder of the TORCS source code, change the following 2 files:

    src/libs/raceengineclient/raceengine.cpp
    
    src/drivers/scr_server/scr_server.cpp
    
    
Afterwards, compile and install again TORCS:

    if it has been already installed, then running the followings commands should be enough:
    
            ./configure
            make
            sudo make install    
    
    otherwise, follow the installation procedure described on Blackboard
